# dragenli-ui

dragenli-ui is a ui for Vue.js.

## Install

```bash
$ npm install dragenli-ui
```

## Import

### Import using module

### Import using script tag

``` html
<script src="../node-modules/dragenli-ui/dist/dragenli-ui.js"></script>
```
